# SparkLLM

>[SparkLLM](https://xinghuo.xfyun.cn/spark) is a large-scale cognitive model independently developed by `iFLYTEK`.
>It has cross-domain knowledge and language understanding ability by learning a large amount of texts, codes and images.
>It can understand and perform tasks based on natural dialogue.

## Installation and Setup

Get SparkLLMâ€™s `app_id`, `api_key` and `api_secret` from [iFlyTek SparkLLM API Console](https://console.xfyun.cn/services/bm3) 
(for more info, see [iFlyTek SparkLLM Intro](https://xinghuo.xfyun.cn/sparkapi)), then set environment variables 
`IFLYTEK_SPARK_APP_ID`, `IFLYTEK_SPARK_API_KEY` and `IFLYTEK_SPARK_API_SECRET`.
